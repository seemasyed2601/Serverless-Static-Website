// About This Project

What Is a Serverless Static Site?

A serverless static website serves pre-built HTML, CSS, and JavaScript files directly from object storage like Amazon S3, delivered globally via Cloudflare CDN — no web server, no runtime, no maintenance overhead.

Files are cached at edge locations worldwide, giving every visitor near-instant load times regardless of geography.

  • ✦ No server management or patching
  • ✦ Scales automatically to millions of requests
  • ✦ SSL/HTTPS included via Cloudflare — free
  • ✦ CI/CD via AWS CodePipeline + GitHub
