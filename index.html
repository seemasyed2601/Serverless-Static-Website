// About This Project

What Is a Serverless Static Site?

A serverless static website serves pre-built HTML, CSS, and JavaScript files directly from object storage like Amazon S3, distributed globally via CloudFront CDN — no web server, no runtime, no maintenance overhead.

Files are cached at edge locations worldwide, giving every visitor near-instant load times regardless of geography.

  • ✦ No server management or patching
  • ✦ Scales automatically to millions of requests
  • ✦ SSL/HTTPS included via ACM
  • ✦ Custom domain via Route 53

// How to Deploy

Up in 4 Steps

01

Create S3 Bucket

Name it after your domain, enable static website hosting, set index.html as the root document.

→
02

Upload Files

Upload your entire project folder. Set public-read permissions or use a bucket policy.

→
03

Create CloudFront Distribution

Point it to your S3 bucket. Set default root object to index.html.

→
04

Go Live

Use your CloudFront URL or configure a custom domain via Route 53.
